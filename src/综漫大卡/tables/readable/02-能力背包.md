# 能力背包

- uid: `sheet_ability_backpack`
- orderNo: 2

## 说明

- 记录主角通过抽卡系统获得的所有能力。一行一个能力。
- - 列1: 能力名称 - 抽到的能力名（全表唯一）
- - 列2: 稀有度 - N/R/SR/SSR
- - 列3: 分类 - 幸运/厄运/物品/事件/角色/无事
- - 列4: 效果描述 - 该能力的具体效果说明
- - 列5: 获得时间 - 日期 YYYY-MM-DD

## 更新规则

- 初始化: 故事初始化时背包为空，暂不插入。
- 更新: 一般不更新已有能力。SQL示例: UPDATE ability_backpack SET 效果描述 = '新描述' WHERE 能力名称 = '某能力';
- 插入: 玩家抽卡获得新能力时添加。SQL示例: INSERT INTO ability_backpack (row_id, 能力名称, 稀有度, 分类, 效果描述, 获得时间) VALUES ((SELECT COALESCE(MAX(row_id), 0) + 1 FROM ability_backpack), '雨中偶遇', 'R', '事件', '下雨天和喜欢的人恰好同路', '2012-04-09');
- 删除: 能力被消耗或失效时删除。SQL示例: DELETE FROM ability_backpack WHERE 能力名称 = '一次性幸运符';

## DDL

```sql
CREATE TABLE ability_backpack ( -- 能力背包
  row_id INTEGER PRIMARY KEY, -- 行号
  ability_name TEXT NOT NULL UNIQUE, -- 能力名称
  rarity TEXT NOT NULL DEFAULT 'N' CHECK(rarity IN ('N', 'R', 'SR', 'SSR')), -- 稀有度
  category TEXT NOT NULL CHECK(category IN ('幸运', '厄运', '物品', '事件', '角色', '无事')), -- 分类
  effect TEXT NOT NULL DEFAULT '', -- 效果描述
  obtained_time TEXT NOT NULL DEFAULT '' -- 获得时间
);
```
