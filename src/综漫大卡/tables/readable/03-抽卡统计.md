# 抽卡统计

- uid: `sheet_gacha_stats`
- orderNo: 3

## 说明

- 记录主角抽卡的累积统计数据。此表有且仅有一行。
- - 列1: 总抽数 - 累积抽卡总次数
- - 列2: N次数 - 抽到N级的次数
- - 列3: R次数 - 抽到R级的次数
- - 列4: SR次数 - 抽到SR级的次数
- - 列5: SSR次数 - 抽到SSR级的次数

## 更新规则

- 初始化: 故事初始化时插入一条记录，所有计数初始为0。
- 更新: 每次抽卡后更新。SQL示例: UPDATE gacha_stats SET 总抽数 = 总抽数 + 1, N次数 = N次数 + 1 WHERE row_id = 1;
- 插入: 禁止操作。
- 删除: 禁止删除。

## DDL

```sql
CREATE TABLE gacha_stats ( -- 抽卡统计
  row_id INTEGER PRIMARY KEY, -- 行号
  total_draws INTEGER NOT NULL DEFAULT 0, -- 总抽数
  n_count INTEGER NOT NULL DEFAULT 0, -- N次数
  r_count INTEGER NOT NULL DEFAULT 0, -- R次数
  sr_count INTEGER NOT NULL DEFAULT 0, -- SR次数
  ssr_count INTEGER NOT NULL DEFAULT 0 -- SSR次数
);
```
