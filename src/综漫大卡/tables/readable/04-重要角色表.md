# 重要角色表

- uid: `sheet_important_characters`
- orderNo: 4

## 说明

- 记录故事中的重要角色及其好感度。一行一个角色。
- - 列1: 姓名 - 角色名（全表唯一）
- - 列2: 性别年龄 - 大致年龄
- - 列3: 简介 - 一句话（<=15字）
- - 列4: 外貌 - 外貌特征
- - 列5: 关键物品 - 用分号分隔
- - 列6: 是否离场 - 是/否
- - 列7: 过去经历 - 背景故事（<=300字）
- - 列8: 好感度 - 0~500

## 更新规则

- 初始化: 添加所有已知角色。好感度：陌生人0~10，同学10~30，朋友30~60，暗恋60~90。
- 更新: 互动后更新好感度。单次±15，重大事件±30。SQL示例: UPDATE important_characters SET 好感度 = 好感度 + 10 WHERE 姓名 = '雪之下雪乃';
- 插入: 新角色出场时添加。
- 删除: 角色离场时设是否离场=是，一般不删除。

## DDL

```sql
CREATE TABLE important_characters ( -- 重要角色表
  row_id INTEGER PRIMARY KEY, -- 行号
  name TEXT NOT NULL UNIQUE, -- 姓名
  gender_age TEXT, -- 性别年龄
  brief_intro TEXT CHECK(brief_intro IS NULL OR LENGTH(brief_intro) <= 15), -- 简介
  appearance TEXT, -- 外貌
  key_items TEXT, -- 关键物品
  is_absent TEXT NOT NULL DEFAULT '否' CHECK(is_absent IN ('是', '否')), -- 是否离场
  past_experience TEXT CHECK(past_experience IS NULL OR LENGTH(past_experience) <= 300), -- 过去经历
  affection INTEGER NOT NULL DEFAULT 0 CHECK(affection >= 0 AND affection <= 500) -- 好感度
);
```
