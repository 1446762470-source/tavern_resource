# 主角信息表

- uid: `sheet_protagonist_info`
- orderNo: 1

## 说明

- 记录{{user}}的基本信息、班级、自我介绍和金币余额。此表有且仅有一行。
- - 列1: 姓名 - {{user}}的角色名
- - 列2: 性别 - 男或女
- - 列3: 班级 - 所在的班级
- - 列4: 自我介绍 - 一句话自我介绍
- - 列5: 余额 - 当前金币数，用于抽卡消费

## 更新规则

- 初始化: 故事初始化时插入一条记录。余额初始为50000。
- 更新: 余额根据抽卡消费和金币获取更新。SQL示例: UPDATE protagonist_info SET 余额 = 余额 - 20000 WHERE row_id = 1;
- 插入: 禁止操作。此表只能有一行。
- 删除: 禁止删除。

## DDL

```sql
CREATE TABLE protagonist_info ( -- 主角信息表
  row_id INTEGER PRIMARY KEY, -- 行号
  name TEXT NOT NULL DEFAULT '', -- 姓名
  gender TEXT NOT NULL DEFAULT '男' CHECK(gender IN ('男', '女')), -- 性别
  class TEXT NOT NULL DEFAULT '', -- 班级
  self_intro TEXT NOT NULL DEFAULT '', -- 自我介绍
  balance INTEGER NOT NULL DEFAULT 50000 CHECK(balance >= 0) -- 余额
);
```
